# 💎 Diamond Analytics | Machine Learning

## 📌 Descripción del Proyecto
Diamond Analytics es una aplicación interactiva desarrollada con **Streamlit**, que permite analizar y predecir el precio y la calidad del corte de los diamantes utilizando modelos de **Machine Learning**.

Los modelos de **Regresión** y **Clasificación** han sido entrenados previamente y se cargan automáticamente desde **Google Drive** si no están disponibles localmente.

---
## 🚀 Características Principales
✅ **Análisis Exploratorio de Datos (EDA)**: Visualizaciones interactivas y estadísticas clave.
✅ **Predicción de Precio**: Basado en características físicas y cualitativas del diamante.
✅ **Clasificación del Corte**: Identificación del tipo de corte utilizando técnicas de Machine Learning.
✅ **Simulador Inteligente**: Predicción combinada de precio y calidad del corte.
✅ **Interfaz Intuitiva y Profesional**: Diseño atractivo con gráficos interactivos.

## 🛠️ Instalación y Configuración

### 1️⃣ **Clonar el repositorio**
```bash
git clone [https://github.com/Rafael-TF/rafael_ejercicios.git](https://github.com/Rafael-TF/Machine-learning-Diamonds.git)
cd /Machine-learning-Diamonds
```

### 2️⃣ **Instalar las dependencias**

### 3️⃣ **Ejecutar la aplicación**
